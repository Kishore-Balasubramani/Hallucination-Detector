# LLM Hallucination Detection System

## Overview

This project detects hallucinated responses generated by Large Language Models (LLMs) using the TruthfulQA benchmark dataset.

## Dataset

* TruthfulQA
* 817 benchmark questions
* Balanced evaluation dataset containing truthful and hallucinated responses

## Methodology

### Baseline

* SentenceTransformer Embeddings
* Random Forest Classifier

Result:

* Accuracy: 45%

### Similarity-Based Detector

* Cosine similarity between candidate answers and ground-truth answers

Result:

* Accuracy: 86%

### Final Retrieval-Based Detector

* Compare candidate answer against:

  * Verified Correct Answers
  * Verified Incorrect Answers
* Predict the class with the higher semantic similarity

Result:

* Accuracy: 90%
* F1 Score: 0.90

## Technologies

* Python
* Hugging Face Datasets
* SentenceTransformers
* Scikit-Learn
* TruthfulQA

## Key Finding

Comparing candidate answers against both correct and incorrect reference answers provides significantly better hallucination detection than standard text classification approaches.
